🟢 ধাপ ১ — একদম শুরু (Total Beginner)

"TypeScript কী এবং কেন শিখব? একদম সহজ ভাষায় বোঝাও"
"TypeScript ইন্সটল করা থেকে প্রথম কোড রান করা পর্যন্ত হাতে ধরে শেখাও"
"string, number, boolean, array, tuple, any — সব type উদাহরণ দিয়ে শেখাও"
"Type annotation কী এবং কোথায় লিখতে হয়?"


🟡 ধাপ ২ — ভিত্তি তৈরি (Foundation)

"Interface আর Type alias কী — কখন কোনটা ব্যবহার করব?"
"Union type, optional type, আর default value কীভাবে কাজ করে?"
"TypeScript-এ function কীভাবে লেখে — সব ধরনের উদাহরণ দাও"
"Enum কী এবং কখন ব্যবহার করব?"
"Array আর Object কীভাবে type করে TypeScript-এ?"


🔵 ধাপ ৩ — মধ্যম স্তর (Intermediate)

"Generics কী — বাস্তব উদাহরণ দিয়ে বোঝাও"
"TypeScript-এ Class, constructor, inheritance, access modifier শেখাও"
"Type guard আর type narrowing কীভাবে কাজ করে?"
"Utility types — Partial, Pick, Omit, Record, Readonly উদাহরণসহ শেখাও"
"Async/await আর Promise TypeScript-এ কীভাবে type করে?"
"tsconfig.json-এর গুরুত্বপূর্ণ সেটিংস কী কী এবং কেন দরকার?"


🟠 ধাপ ৪ — অ্যাডভান্সড (Advanced)

"Conditional type আর mapped type কী — কখন ব্যবহার করব?"
"Decorators কী এবং কীভাবে কাজ করে TypeScript-এ?"
"Module system — import/export TypeScript-এ কীভাবে কাজ করে?"
"Declaration file (.d.ts) কী এবং কখন বানাতে হয়?"
"Third-party library-তে @types কীভাবে ব্যবহার করব?"
"Namespace আর Module-এর পার্থক্য কী?"

✅ React + TypeScript

"React-এ TypeScript কীভাবে ব্যবহার করে — props, state, hooks সব type করা শেখাও"
"React component-এ generic props কীভাবে লেখে?"
"Custom hook TypeScript-এ কীভাবে বানাই?"
